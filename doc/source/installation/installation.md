## Command Line


## Python

Neug python package are availabel at `pypi`, use `pip` to install the python package on Linux and macos. 

````{note}
Currently only support on python >= 3.8. For macos, we require minimum version of 10.9.
````

```bash
pip3 install neug
# Specify the index-url 
pip3 install neug -i https://pypi.org/simple
# Aliyun mirror
pip3 install neug -i https://mirrors.aliyun.com/pypi/simple/
```
