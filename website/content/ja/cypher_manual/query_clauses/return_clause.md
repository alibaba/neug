# Return 句

Return と With は似た機能を提供し、どちらもデータのさらなる集約や射影に使用されます。違いは、Return は処理された結果を返して表示する必要がある点です。ここでは Return の機能自体について詳しく説明しません。使い方は [With 句](with_clause.md) のセクションを参照してください。ここでは主に結果の出力と一般的な Return の使用パターンに焦点を当てます。

## Return ノード

### 単一ラベルを持つノードを返す
```
Match (a:person) Return a;
```

出力には各personノードの内部ID（グラフデータベースによって割り当てられたもの）、ラベル、およびすべてのプロパティが表示されます：
```
+-------------------------------------------------------+
| a                                                     |
+=======================================================+
| {_ID: 0, _LABEL: person, name: marko, age: 29} |
+-------------------------------------------------------+
| {_ID: 1, _LABEL: person, name: vadas, age: 27} |
+-------------------------------------------------------+
| {_ID: 2, _LABEL: person, name: josh, age: 32}  |
+-------------------------------------------------------+
| {_ID: 3, _LABEL: person, name: peter, age: 35} |
+-------------------------------------------------------+
```

### 複数のラベルを持つノードを返す
```
Match (a) Return a;
```

出力には各ノードの内部ID、ラベル、およびすべてのプロパティがそれぞれのノードタイプで表示されます：
```
+-----------------------------------------------------------------------------+
| a                                                                           |
+=============================================================================+
| {_ID: 0, _LABEL: person, name: marko, age: 29}                       |
+-----------------------------------------------------------------------------+
| {_ID: 1, _LABEL: person, name: vadas, age: 27}                       |
+-----------------------------------------------------------------------------+
| {_ID: 2, _LABEL: person, name: josh, age: 32}                        |
+-----------------------------------------------------------------------------+
| {_ID: 3, _LABEL: person, name: peter, age: 35}                       |
+-----------------------------------------------------------------------------+
| {_ID: 72057594037927936, _LABEL: software, name: lop, lang: java}    |
+-----------------------------------------------------------------------------+
| {_ID: 72057594037927937, _LABEL: software, name: ripple, lang: java} |
+-----------------------------------------------------------------------------+
```


出力には、リレーションシップの内部ID、ラベル、すべてのプロパティ、およびソースノードと宛先ノードのラベルとIDが含まれます：

## Return with TopK

TopK のクエリ結果を出力するために使用される Return、OrderBy、Limit の組み合わせ

```
Match (a:person)-[:knows]->(b:person)
Return a.name, b.name
Order By a.name ASC, b.name ASC
Limit 2;
```

出力:
```
+-------------+-------------+
| _0_a.name   | _2_b.name   |
+=============+=============+
| marko       | josh        |
+-------------+-------------+
| marko       | vadas       |
+-------------+-------------+
```
