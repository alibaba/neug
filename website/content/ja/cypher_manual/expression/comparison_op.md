# 比較演算子

Neug がサポートする各種比較演算子を以下の表で示します。

演算子 | 説明 | 例 | 結果
---------|-------------|---------|--------
`=` | 等しい | `1 = 1` | `true`
`<>` | 等しくない | `1 <> 2` | `true`
`<` | より小さい | `1 < 2` | `true`
`<=` | 以下 | `1 <= 1` | `true`
`>` | より大きい | `3 > 2` | `true`
`>=` | 以上 | `3 >= 3` | `true`

## NULL 値の扱い

Neug は、SQL 標準に従って比較操作における NULL 値を処理します。比較操作のいずれかのオペランドが NULL である場合、使用される比較演算子に関係なく、結果は常に NULL になります。この動作は「三値論理 (three-valued logic)」と呼ばれ、比較の結果は `true`、`false`、または `NULL` のいずれかになります。

次の表は、比較操作における NULL 値の扱いを示しています：

Operator | Example | Result
---------|---------|--------
`=` | `NULL = NULL`, `NULL = 1` | `NULL`
`<>` | `NULL <> NULL`, `NULL <> 1` | `NULL`
`<` | `NULL < NULL`, `NULL < 1` | `NULL`
`<=` | `NULL <= NULL`, `NULL <= 1` | `NULL`
`>` | `NULL > NULL`, `NULL > 1` | `NULL`
`>=` | `NULL >= NULL`, `NULL >= 1` | `NULL`
