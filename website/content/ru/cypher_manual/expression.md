# Expression

## Обзор

Expressions являются фундаментальными компонентами в Cypher, которые позволяют вычислять значения, выполнять вычисления и манипулировать данными в запросах. В отличие от традиционного SQL, который фокусируется на операциях с реляционными данными, expressions в Cypher специально разработаны для графовых структур данных и предоставляют мощные возможности для обхода связей, сопоставления паттернов и графовых вычислений.

Cypher expressions выполняют несколько ключевых задач:

- **Вычисление значений**: Расчет производных значений из свойств узлов, атрибутов связей или вычисленных результатов
- **Оценка условий**: Создание булевых выражений для фильтрации узлов, связей или путей
- **Трансформация данных**: Преобразование и форматирование типов данных для отображения или дальнейшей обработки
- **Обход графа**: Навигация по графовым структурам с использованием path expressions и pattern matching
- **Агрегация**: Выполнение вычислений над коллекциями узлов или связей
- **Обработка строк и текста**: Манипуляции с текстовыми данными для поиска, отображения или анализа

Cypher expressions строятся из двух основных категорий компонентов:

1. **Операторы**: Символы и ключевые слова, которые выполняют базовые операции над операндами
2. **Функции**: Предопределенные операции, инкапсулирующие сложную логику и принимающие параметры

Эти компоненты работают вместе, создавая мощные expressions, которые могут обрабатывать как простые вычисления, так и сложные графовые операции.

## Операторы

Операторы в Neug — это символы или ключевые слова, которые выполняют операции над операндами. Они являются фундаментальными строительными блоками для создания выражений и запросов.

## Функции

Функции в Neug — это заранее определенные операции, которые принимают входные параметры и возвращают вычисленные значения. Они предоставляют специализированный функционал для манипуляций с данными и их анализа.

### Поддерживаемые категории функций

| Категория | Описание |
|----------|-------------|
| [Агрегатные](./expression/agg_func.md) | Функции, которые работают с наборами значений и возвращают один результат (например, `COUNT`, `SUM`, `AVG`, `MAX`, `MIN`) |
| [Приведение типов](./expression/cast_func.md) | Функции для конвертации данных между различными форматами |
| [Временные](./expression/temporal_func.md) | Функции для работы с датами и временем |
| [Графовые паттерны](./expression/graph_func.md) | Функции, предназначенные для работы с узлами, связями или путями |
