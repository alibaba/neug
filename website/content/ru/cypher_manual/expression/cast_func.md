# Функция CAST

Функция CAST主要用于不同数据类型之间的转换。目前我们仅支持字面常量值的类型转换，未来将进一步支持变量、属性以及更复杂的表达式。

## Приведение литералов

Neug на данный момент поддерживает следующие преобразования типов:
- Взаимное преобразование между числовыми типами. К числовым типам относятся: INT32, UINT32, INT64, UINT64, FLOAT, DOUBLE. Преобразование возможно между любыми двумя числовыми типами. В случае переполнения будет выброшено исключение. В таблице ниже указаны возможные исключения.
- Преобразование между String и числовыми типами. String может быть преобразован в любой числовой тип и наоборот. Если конкретное значение не может быть преобразовано, будет выброшено исключение.

| Из \ В | INT32  | UINT32 | INT64  | UINT64 | FLOAT  | DOUBLE |
|--------|--------|--------|--------|--------|--------|--------|
| **INT32**  | ✅ Безопасно | ⚠️ Может произойти переполнение (если значение < 0)        | ✅ Безопасно | ⚠️ Может произойти переполнение (если значение < 0)        | ✅ Безопасно | ✅ Безопасно |
| **UINT32** | ⚠️ Может произойти переполнение (если значение > INT32_MAX)         | ✅ Безопасно | ✅ Безопасно | ✅ Безопасно | ✅ Безопасно | ✅ Безопасно |
| **INT64**  | ⚠️ Может произойти переполнение (если значение < INT32_MIN или > INT32_MAX) | ⚠️ Может произойти переполнение (если значение < 0 или > UINT32_MAX) | ✅ Безопасно | ⚠️ Может произойти переполнение (если значение < 0)        | ✅ Безопасно | ✅ Безопасно |
| **UINT64** | ⚠️ Может произойти переполнение (если значение > INT32_MAX)         | ⚠️ Может произойти переполнение (если значение > UINT32_MAX)       | ⚠️ Может произойти переполнение (если значение > INT64_MAX) | ✅ Безопасно | ✅ Безопасно | ✅ Безопасно |
| **FLOAT**  | ⚠️ Может произойти переполнение (если значение < INT32_MIN или > INT32_MAX) | ⚠️ Может произойти переполнение (если значение < 0 или > UINT32_MAX) | ⚠️ Может произойти переполнение (если значение < INT64_MIN или > INT64_MAX) | ⚠️ Может произойти переполнение (если значение < 0 или > UINT64_MAX) | ✅ Безопасно | ✅ Безопасно |
| **DOUBLE** | ⚠️ Может произойти переполнение (если значение < INT32_MIN или > INT32_MAX) | ⚠️ Может произойти переполнение (если значение < 0 или > UINT32_MAX) | ⚠️ Может произойти переполнение (если значение < INT64_MIN или > INT64_MAX) | ⚠️ Может произойти переполнение (если значение < 0 или > UINT64_MAX) | ⚠️ Может произойти переполнение (если значение < -FLT_MAX или > FLT_MAX) | ✅ Безопасно |
