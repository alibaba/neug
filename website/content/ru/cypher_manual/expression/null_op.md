# Оператор Null

Neug поддерживает операторы `IS NULL` и `IS NOT NULL` для фильтрации значений NULL, которые возникают во время вычислений.

Оператор | Описание
---------|------------
IS NULL | возвращает true, если вычисленное значение равно null
IS NOT NULL | возвращает true, если вычисленное значение не равно null
