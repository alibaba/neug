# Temporal Functions

Temporal Functions — это специализированный набор функций, предназначенных для работы с типами данных даты и интервала. Они предоставляют возможности для создания типов даты/интервала из строковых литералов и извлечения определённых полей из временных значений. В следующей таблице перечислены доступные функции и примеры их использования.

## Справочник функций

| Функция | Описание | Пример | Тип возвращаемого значения |
|----------|-------------|---------|-------------|
| `DATE()` | Создает значение типа `DATE` из строкового литерала | `DATE('2012-01-01')` | `DATE` |
| `TIMESTAMP()` | Создает значение типа `TIMESTAMP` из строкового литерала | `TIMESTAMP('1926-11-21 13:22:19')` | `TIMESTAMP` |
| `INTERVAL()` | Создает значение типа `INTERVAL` из строкового литерала | `INTERVAL('3 DAYS')` | `INTERVAL` |
| `date_part(part, date)` | Извлекает определенную часть из значения типа `DATE` | `date_part('year', DATE('1995-11-02'))` | `INTEGER` |
| `date_part(part, timestamp)` | Извлекает определенную часть из значения типа `TIMESTAMP` | `date_part('month', TIMESTAMP('1926-11-21 13:22:19'))` | `INTEGER` |
| `date_part(part, interval)` | Извлекает определенную часть из значения типа `INTERVAL` | `date_part('days', INTERVAL('1 days'))` | `INTEGER` |
