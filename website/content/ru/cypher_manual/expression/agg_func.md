# Агрегатные функции

Агрегатные функции в основном используются для группировки текущих данных и выполнения агрегатных операций над элементами внутри каждой группы, в конечном итоге выдавая только одно значение для каждой группы. Поддерживаемые Neug агрегатные функции следующие:

Функция | Описание | Может использоваться с DISTINCT | Пример
--------|----------|-------------------------------|-------
count | возвращает количество строк | ДА | Return count(a.name);
collect | собирает элементы в один список | ДА | Return collect(a.name);
min | возвращает минимальное значение | НЕТ | Return min(a.age);
max | возвращает максимальное значение | НЕТ | Return max(a.age);
sum | суммирует значения | НЕТ | Return sum(a.age);
avg | возвращает среднее значение | НЕТ | Return avg(a.age);
