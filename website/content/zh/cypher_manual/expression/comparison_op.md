# 比较运算符

我们通过下表展示 Neug 支持的各种比较运算符。

运算符 | 描述 | 示例 | 结果
---------|-------------|---------|--------
`=` | 等于 | `1 = 1` | `true`
`<>` | 不等于 | `1 <> 2` | `true`
`<` | 小于 | `1 < 2` | `true`
`<=` | 小于或等于 | `1 <= 1` | `true`
`>` | 大于 | `3 > 2` | `true`
`>=` | 大于或等于 | `3 >= 3` | `true`

## NULL 值处理

Neug 在比较操作中按照 SQL 标准处理 NULL 值。当比较操作中的任意操作数为 NULL 时，无论使用何种比较运算符，结果始终为 NULL。这种行为被称为"三值逻辑"，即比较结果可以是 `true`、`false` 或 `NULL`。

下表展示了在比较操作中 NULL 值的处理方式：

运算符 | 示例 | 结果
---------|---------|--------
`=` | `NULL = NULL`, `NULL = 1` | `NULL`
`<>` | `NULL <> NULL`, `NULL <> 1` | `NULL`
`<` | `NULL < NULL`, `NULL < 1` | `NULL`
`<=` | `NULL <= NULL`, `NULL <= 1` | `NULL`
`>` | `NULL > NULL`, `NULL > 1` | `NULL`
`>=` | `NULL >= NULL`, `NULL >= 1` | `NULL`
