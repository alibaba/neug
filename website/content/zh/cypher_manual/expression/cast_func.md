# Cast Function

CAST 函数主要用于在不同数据类型之间进行转换。我们目前仅支持对字面常量值进行类型转换，未来将进一步支持变量、属性以及更复杂的表达式。

## 类型转换字面量

Neug 目前支持以下类型转换：

- **Numeric 类型之间的相互转换**。Numeric 类型包括：`INT32`、`UINT32`、`INT64`、`UINT64`、`FLOAT`、`DOUBLE`。任意两个 Numeric 类型之间都可以进行转换。如果发生溢出，则会抛出异常。下表展示了可能出现的异常情况。

- **String 与 Numeric 类型之间的转换**。String 可以与任意 Numeric 类型相互转换。如果具体值无法转换，则会抛出异常。

| From \ To | INT32  | UINT32 | INT64  | UINT64 | FLOAT  | DOUBLE |
|-----------|--------|--------|--------|--------|--------|--------|
| **INT32**     | ✅ 安全 | ⚠️ 可能溢出（如果值 < 0）               | ✅ 安全 | ⚠️ 可能溢出（如果值 < 0）               | ✅ 安全 | ✅ 安全 |
| **UINT32**    | ⚠️ 可能溢出（如果值 > INT32_MAX）            | ✅ 安全 | ✅ 安全 | ✅ 安全 | ✅ 安全 | ✅ 安全 |
| **INT64**     | ⚠️ 可能溢出（如果值 < INT32_MIN 或 > INT32_MAX） | ⚠️ 可能溢出（如果值 < 0 或 > UINT32_MAX）      | ✅ 安全 | ⚠️ 可能溢出（如果值 < 0）               | ✅ 安全 | ✅ 安全 |
| **UINT64**    | ⚠️ 可能溢出（如果值 > INT32_MAX）            | ⚠️ 可能溢出（如果值 > UINT32_MAX）             | ⚠️ 可能溢出（如果值 > INT64_MAX）        | ✅ 安全 | ✅ 安全 | ✅ 安全 |
| **FLOAT**     | ⚠️ 可能溢出（如果值 < INT32_MIN 或 > INT32_MAX） | ⚠️ 可能溢出（如果值 < 0 或 > UINT32_MAX）      | ⚠️ 可能溢出（如果值 < INT64_MIN 或 > INT64_MAX） | ⚠️ 可能溢出（如果值 < 0 或 > UINT64_MAX） | ✅ 安全 | ✅ 安全 |
| **DOUBLE**    | ⚠️ 可能溢出（如果值 < INT32_MIN 或 > INT32_MAX） | ⚠️ 可能溢出（如果值 < 0 或 > UINT32_MAX）      | ⚠️ 可能溢出（如果值 < INT64_MIN 或 > INT64_MAX） | ⚠️ 可能溢出（如果值 < 0 或 > UINT64_MAX） | ⚠️ 可能溢出（如果值 < -FLT_MAX 或 > FLT_MAX） | ✅ 安全 |
